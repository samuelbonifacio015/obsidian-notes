# Extraction Rules

## Prioridad De Fuentes

1. Check-in completado.
2. Todoreviewer.
3. Summary diario.
4. Todolist.
5. Plan semanal/mensual.
6. Inferencia desde contexto historico.

## Contradicciones

- Si el todolist dice completado pero el check-in contradice, prioriza el check-in.
- Si el reviewer identifica una bandera de riesgo, subela al tracker semanal.
- Si una tarea aparece pendiente 2+ dias, clasificala como riesgo activo.
- Si una actividad fue mencionada sin evidencia concreta, marcala como avance parcial.

## Scoring

- 80-100%: semana productiva y bien cerrada.
- 60-79%: semana intensa/parcial, con riesgos claros.
- 40-59%: semana irregular.
- 0-39%: semana floja o descarrilada.

## Reglas Por Area

- Academico pesa mas durante semanas con examen o entrega UPC.
- MaquinariasJyS pesa mas si hay deadline de cliente o avance frontend.
- Personal no se ignora: sueno, pasos y energia afectan la lectura real del rendimiento.
